# 🍗 KFC Order Management Bot

Sistema de gestión de órdenes KFC con soporte multiplataforma (Windows/Linux) para Telegram.

## ✨ Características

- ✅ Detección automática de SO por tienda (Windows/Linux)
- ✅ Generación de imágenes de facturas y comandas multiplataforma
- ✅ Sistema de impresión dinámico con lógica alternativa
- ✅ Reportes con gráficos automáticos
- ✅ Gestión de logs organizados por año/mes/día
- ✅ Interface con botones para fácil uso
- ✅ Cierre automático por inactividad
- ✅ Soporte para tiendas K001 a K999
